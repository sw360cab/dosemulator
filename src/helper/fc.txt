Compares two files and displays the differences between
them


FC [\C] [\N] [path1]filename1 [path2]filename2

  \C         Disregards the case of letters.
  \N         Displays the line numbers on an ASCII comparison.
  [path1]filename1
             Specifies the first file to compare.
  [path2]filename2
             Specifies the second file to compare.
